# Angular Basics: How To Get the Value of a Selected Dropdown Menu Item

## Project Summary
A comparison of different methods to retrieve selected values from dropdown menus in Angular, including Template Reference Variables, ViewChild, and ngModel.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Interact with the dropdowns and observe the console/UI updates for each method.
